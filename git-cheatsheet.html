📘 Git Ultimate Cheatsheet (with Graphs & Explanations)

🔧 Configuration

git config user.name "Your Name"
git config user.email "you@example.com"
git config init.defaultBranch=main

Sets your identity and default branch name

🏗️ Initialize a Repository

git init

Creates a new Git repo in the current directory.

📝 Track and Stage Files

git status
git add .
git add <file>
git diff
git diff --cached

♻️ Restore / Unstage / Remove Files

git restore --staged <file>
git rm --cached <file>
git rm <file>

🔀 Renaming or Moving Files

git mv old.txt new.txt

✅ Committing

git commit -m "Message"
git commit -am "Quick commit"

🕵️‍♂️ View Logs & Details

git log
git log --oneline
git log --graph --decorate
git show <commit>
cat .git/HEAD

🌿 Branching

git branch <branch>
git checkout <branch>
git checkout -b <branch>
git branch
git branch -d <branch>

🍒 Cherry Pick

git cherry-pick <commit>

Use to apply a single commit from one branch to another

🔀 Merging

git merge <branch>
git merge --abort

🚀 Rebase

git rebase <branch>
git rebase -i HEAD~3

🌿 Tags

git tag
git tag v1.0
git tag -a v1.0 -m "Release v1.0"
git show v1.0
git push origin v1.0
git push origin --tags

🔥 Undoing Commits

git reset --soft HEAD~1
git reset --mixed HEAD~1
git reset --hard HEAD~1

🔎 Git Ignore

*.log
node_modules/
.env

⚙️ Git Attributes

*.sh text eol=lf
*.jpg binary
